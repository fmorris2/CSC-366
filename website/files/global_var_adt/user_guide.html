Overview

The global variable abstract data type (ADT) currently offers an easy way to do basic arithmetic operations on defined variables. These variables are stored globally, so they can be referenced throughout the lifetime of the Prolog session.

The global variable ADT currently supports non-destructive operations on variables. This means that the input variables will not be changed, and you must provide the name of the output variable where the result will be stored. The global variable ADT currently supports the following operations:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Raising a number to an exponent

    • Usage

    The global variable ADT provides some simple rules to make declaring and performing operations on variables easy.

    • declare(Var,Val)
      • Binds Val to the variable named Var
      • Overrides the current binding to Var if it is already defined
    • valueOf(Variable,Value)
      • Finds the value of Variable in our current bindings, and returns it as Value
    • undeclare(Var)
      • Removes any binding in the ADT with the name Var
    • displayBindings
      • Lists all current bindings in the ADT
    • inc(Variable)
      • Adds 1 to the current value mapped to Variable
    • dec(Variable)
      • Subtracts 1 from the current value mapped to Variable
    • add(FirstVar,SecondVar,ResultVar)
      • Adds the value bound to SecondVar to the value bound to FirstVar, and binds the result to a variable named ResultVar
    • sub(FirstVar,SecondVar,ResultVar)
      • Subtracts the value bound to SecondVar from the value bound to FirstVar, and binds the result to a variable named ResultVar
    • mul(FirstVar,SecondVar,ResultVar)
      • Multiplies the value bound to FirstVar with the value bound to SecondVar, and binds the result to a variable named ResultVar
    • div(FirstVar,SecondVar,ResultVar)
      • Divides the value bound to FirstVar with the value bound to SecondVar, and binds the result to a variable named ResultVar
    • pow(FirstVar,SecondVar,ResultVar)
      • Raises the value bound to FirstVar by the exponent represented by the value bound to SecondVar, and binds the result to a variable named ResultVar
